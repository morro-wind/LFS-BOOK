6.48. Libelf 0.170

Libelf 是处理 ELF（可执行与可链接格式）文件的库。

大致构建用时: 0.6 SBU
所需磁盘空间: 74 MB

6.48.1. 安装 Libelf

Libelf 是软件包 elfutils-0.170 种的一部分。使用 elfutils-0.170.tar.xz 作为源码包。

编译 Libelf 前的准备： 

./configure --prefix=/usr

编译软件包： 

make

测试结果，请输入 make check 查看。

仅安装 Libelf： 

make -C libelf install
install -vm644 config/libelf.pc /usr/lib/pkgconfig
